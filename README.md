# YouTube Tutorial Summarizer

Transform any YouTube tutorial into actionable guides, summaries, and key concepts.

## 🚀 Features
- Extract transcripts from any YouTube video
- AI-powered summaries (requires OpenAI API key)
- Step-by-step instruction extraction
- Key concepts identification
- Clean, modern UI

## 🛠️ Tech Stack
- **Backend**: FastAPI + Langflow + YouTube Transcript API
- **Frontend**: HTML, CSS, JavaScript
- **AI**: OpenAI GPT models
